Getting Started

This is an api to create database from csv file and returns

## **Installation**
1. Install dependencies:
```
pipenv install
```
2. ACTIVATE envirtument:
```
pipenv shell
```
3. python  data:
```
python manage.py makemigrations
```
3. python  data:
```
python manage.py migrate
```
## **Testing**
```
python manage.py test
```
